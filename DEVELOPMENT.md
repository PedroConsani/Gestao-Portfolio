# Projeto Angular - Gestão de Carteira de Ações

## 📋 Checklist do Desenvolvimento

### Funcionalidades Base
- [x] Estrutura Angular com componentes standalone
- [x] Modelos TypeScript para Stock e Portfolio
- [x] Serviço de API para cotações (StockApiService)
- [x] Serviço de gestão de portfólio (PortfolioService)
- [x] Serviço de armazenamento (StorageService)
- [x] Componente de tabela principal (PortfolioTableComponent)
- [x] Componente de linha de stock (StockRowComponent)
- [x] Componente de sumário (PortfolioSummaryComponent)
- [x] Página principal (PortfolioPageComponent)

### Funcionalidades de Dados
- [x] Carregamento de dados JSON
- [x] Cálculo de totais (quantidade × preço)
- [x] Cálculo de valor actual (quantidade × cotação)
- [x] Cálculo de variação em percentagem
- [x] Integrações de API para cotações
- [x] Persistência em localStorage

### UI/UX
- [x] Tabela responsiva
- [x] Cores para variações (verde/vermelho/preto)
- [x] Formatação de valores monetários
- [x] Botão de actualização de cotações
- [x] Indicador de carregamento
- [x] Mensagens de erro

### Configuração
- [x] Ficheiro de ambiente
- [x] Configuração de API
- [x] Dados de exemplo (portfolio.json)
- [x] Estilos globais

## 🔧 Próximas Etapas

1. Executar `npm install` para instalar dependências
2. Configurar a chave de API em `src/environments/environment.ts`
3. Executar `npm start` para iniciar o servidor
4. Aceder a `http://localhost:4200`
